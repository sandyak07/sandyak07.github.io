Automation Plus

In Development

Freshdesk Custom App

Automation Plus is a configurable Freshdesk custom app designed to add additional workflow safeguards directly within the ticket experience. It helps administrators define rules that can evaluate ticket context and provide agents with clear warnings or confirmations before selected actions continue.

The problem

Standard support workflows sometimes require additional checks that depend on an organisation's operating policies. Agents may need reminders before closing tickets, sending responses, changing important ticket properties, or completing other sensitive actions.

Automation Plus is intended to make these checks clearer and more consistent without forcing teams to rely only on documentation or manual supervision.

Key features

  • Configurable workflow rules — define guidance that reflects your team's operating policies.
  • Ticket-context conditions — evaluate relevant ticket information before a configured workflow continues.
  • Agent warning messages — surface clear, timely reminders where they are most useful.
  • Confirmation before selected actions — give agents a clear choice to continue or return to the ticket.
  • Grouped ALL/ANY logic — combine conditions to express more focused workflow rules.
  • Administrative rule management — keep rule configuration in the hands of the people who manage workflows.
  • Clear validation and error states — make rule feedback understandable when input needs attention.
  • Privacy-conscious design — being designed around data minimisation and least-privilege access.
  • Responsive Freshdesk-aligned interface — designed to remain clear across the ticket experience.

How it works

  1. 1An administrator creates a workflow rule.
  2. 2The rule checks relevant ticket information.
  3. 3Automation Plus displays the configured warning or confirmation.
  4. 4The agent reviews the message and continues or returns to the ticket.

Current status

Automation Plus is currently under active development and validation. Features and supported workflows may change before a public release.

Privacy-conscious design

Automation Plus is being designed around data minimisation and least-privilege access, with the goal of using only the information needed to support configured workflow guidance.
